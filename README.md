# 在线花店后台管理系统 java665

## 项目概述

在线花店后台管理系统是基于SpringBoot与Vue技术构建的，用于实现花店日常业务的管理与自动化。系统包括用户管理、库存管理、销售管理、采购管理以及数据分析等多个功能模块。

## 技术栈

### 前端

- Vue
- Vuex
- Vue Router
- Axios
- Element-ui
- Echarts

### 后端

- Java
- SpringBoot
- Mybatis
- Mybatis-Plus
- jdk8
- Maven
- Mysql

## 功能模块

- **登录功能**：支持管理员、采购员、销售员三种身份的用户登录。
  - 用户管理：管理系统操作者账户。
  - 角色与权限：管理员可以为不同角色分配菜单权限。
- **菜单管理**：管理后台菜单的增删改。
- **供应商管理**：管理供应商信息。
- **客户管理**：管理客户信息。
- **采购与入库**：采购员提交采购申请，管理员审核，通过后进行入库操作。
  - 入库统计：统计入库信息。
- **销售与出库**：销售员提交出库通知，管理员审核，通过后进行出库操作。
  - 出库统计：统计出库信息。
- **花材管理**：管理花材信息。
- **花材类别管理**：管理花材类别信息。
- **单位管理**：管理单位信息。
- **库存盘点**：盘点库存，提醒采购。
- **登录日志**：记录用户登录信息。
- **数据分析**：统计经营数据，如客户数量和热销花卉排行。

## 系统角色

系统定义了以下角色：
- 管理员：负责系统的整体管理与权限分配。
- 采购员：负责采购申请与入库操作。
- 销售员：负责销售订单处理与出库操作。

## 运行环境

- 后端：Java 1.8+，Maven，MySQL 5.7+
- 前端：Node.js，现代浏览器

## 部署步骤

部署步骤将包含以下内容，但具体细节需根据实际环境进行调整：
1. 安装后端依赖环境：Java, Maven, MySQL
2. 构建后端工程，部署至服务器
3. 安装前端依赖环境：Node.js
4. 构建前端工程，部署至Web服务器
